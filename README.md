# AI Engineering Journey (90 Days)

This repository documents my journey to becoming an AI Engineer over a 90-day period.
It contains my daily learning progress, code implementations, and projects.

## Start Date

Day 0 – Research Phase

## Goal

To become proficient and job-ready in AI Engineering by building practical skills through consistent daily learning and hands-on projects.

## Approach

* Study core AI Engineering concepts daily
* Build small, incremental projects to reinforce learning
* Maintain consistency and track progress publicly

## Repository Structure

* Each day has a dedicated folder containing:

  * Code implementations (`.py` files)
  * Notes (`notes.md`)
* Larger projects are organized inside the `projects/` directory

## Progress Tracking

Daily activities and milestones are documented to ensure accountability and measure improvement over time.

## Objective

By the end of this journey, this repository will serve as a complete record of my growth, skills, and projects in AI Engineering.
